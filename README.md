# QuizApp
The app is working almost correctly i.e. has one bug( mentioned below).

###  Features:
1. Fetches questions from an API.
2. Renders the questions dynamically.
3. Shows the result after submission.

### Technologies Used
- React.js
- JavaScript 
- CSS for styling
